# Trinity Dev Web - Frontend

React-based frontend application for the Trinity grocery management system.

## Setup

```bash
npm install
npm run dev
```

## Structure

```
src/
├── components/     # Reusable UI components
├── pages/          # Page components
├── services/       # API integration
├── types/          # TypeScript definitions
├── hooks/          # Custom React hooks
└── utils/          # Utility functions
```

## Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build
- `npm run lint` - Run ESLint

## Features

- JWT Authentication
- Product Management
- Customer Management
- Invoice Processing
- Analytics Dashboard
- Responsive Design

## Technology

- React 18
- TypeScript
- Material-UI
- React Query
- Recharts
- Vite
